# 7-Day Practice Plan (Repertoire)

Date: 2026-02-28
Duration target: 45–75 min/day
Structure: Warmup → Core drill → Song work → Log

## Daily template
- **Warmup (10–15 min):** foundation + breath + gentle resonance activation
- **Core drill (15–25 min):** day-specific focus
- **Song work (15–25 min):** apply drill to song section
- **Log (3–5 min):** what improved + what needs work tomorrow
