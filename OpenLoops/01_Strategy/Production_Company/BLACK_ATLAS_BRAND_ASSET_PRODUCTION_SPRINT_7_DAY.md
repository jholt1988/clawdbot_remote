# Black Atlas Productions — Brand Asset Production Sprint (7 Days)

## Sprint Goal
Ship a coherent, usable v1 brand asset set for decks, one-sheets, social, and outreach.

## Deliverables by Day 7
- 1 master deck theme file
- 3 deck covers (Sponsor/Grant/Internal)
- 6 social tiles (template-based)
- 2 one-sheet templates (Sponsor + Grant)
- 1 mini brand kit folder (colors, type, boilerplate, logo rules)

---

## Day 1 — System Lock
- [ ] Finalize brand guide + visual examples as source of truth
- [ ] Confirm palette hex values and typography pair
- [ ] Create folder structure:
  - `/brand/exports`
  - `/brand/templates`
  - `/brand/social`
  - `/brand/one-sheets`
- [ ] Create one reference board (mood + typography + composition)

## Day 2 — Deck Foundation
- [ ] Build master deck theme (title/content/data slides)
- [ ] Add reusable section divider slide
- [ ] Build KPI card and budget slide style once
- [ ] Apply to Sponsor deck cover + first 3 slides

## Day 3 — Deck Completion
- [ ] Finish Sponsor deck styling
- [ ] Port style to Grant deck
- [ ] Port style to Internal deck
- [ ] Export PDF drafts for all 3 decks

## Day 4 — One-Sheets
- [ ] Build Sponsor one-sheet template
- [ ] Build Grant one-sheet template
- [ ] Add CTA/footer/contact blocks
- [ ] Export print and digital variants

## Day 5 — Social Assets
- [ ] Produce 6 tiles from template pack
- [ ] Generate 2 alternative headline versions for each
- [ ] Export in IG square + story sizes
- [ ] Stage first 2-week content queue

## Day 6 — QA + Consistency Pass
- [ ] Check all docs for metric consistency:
  - 3 events
  - 45 paid roles
  - 900 attendance
  - $18,000 budget
- [ ] Check typography and color consistency
- [ ] Check tagline consistency across files
- [ ] Tighten copy for brevity and impact

## Day 7 — Release Bundle
- [ ] Build final “Brand Assets v1” bundle folder
- [ ] Export editable + PDF + PNG packages
- [ ] Create changelog (`BRAND_ASSETS_V1_CHANGELOG.md`)
- [ ] Prepare next-sprint backlog (logo lock, website hero, motion snippets)

---

## Daily Cadence (recommended)
- 60–90 min focused production block
- 15 min end-of-day review + next-day prep
- Keep scope fixed; no new asset types mid-sprint

## Definition of Done
- Assets are reusable without redesigning each time
- External-facing docs feel visually unified
- Brand signal is clear: raw independent + mythic + editorial
