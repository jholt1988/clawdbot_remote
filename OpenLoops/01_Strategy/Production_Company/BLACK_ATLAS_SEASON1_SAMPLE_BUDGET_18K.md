# Black Atlas Live — Season One Sample Budget ($18,000)

**Program:** 3 large-scale 18+ events in Wichita  
**Budget model:** realistic starter grant + earned revenue scenario

## Expense Budget

| Category | Amount |
|---|---:|
| Performer Fees (headliners + supporting artists) | $5,400 |
| Technical Crew (audio, lighting, stage mgmt) | $2,250 |
| Venue Rental / Venue Costs | $2,100 |
| Production Design (set, décor, props, wardrobe support) | $900 |
| Marketing & Promotion (creative, ads, print) | $1,500 |
| Documentation (photo + highlight video) | $1,200 |
| Insurance / Permits / Licenses | $650 |
| Security / Front-of-House Support | $900 |
| Admin & Ops (software, accounting, transaction costs) | $600 |
| Accessibility / Inclusion Support | $450 |
| Contingency (12.8%) | $2,050 |
| **Total Expenses** | **$18,000** |

## Income Budget

| Revenue Source | Amount |
|---|---:|
| Grant Funding | $10,000 |
| Sponsorships & Donations | $4,000 |
| Ticket Revenue (net) | $3,500 |
| In-Kind Support (valued) | $500 |
| **Total Income** | **$18,000** |

## Per-Event Operating Snapshot (avg.)
- Average event cost: **$6,000**
- Average target attendance: **300**
- Average target gross ticket sales: **$2,000–$2,500** (depending on tiering/capacity)

## Assumptions
1. Three events are staggered to preserve quality and maximize audience growth.
2. Artist pay is protected as a core line item.
3. Documentation is treated as strategic infrastructure (future grants/sponsors/artist portfolios).
4. Contingency is intentionally set above 10% for first-season risk control.

## Notes for Grant Adaptation
- If grant is below request, reduce paid ad spend first, not artist compensation.
- If sponsorship outperforms target, expand documentation and artist pay bands.
- For conservative versions, model ticket net at $2,500 total instead of $3,500 and increase grant ask accordingly.
