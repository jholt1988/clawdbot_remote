# Heartbeat - Proactive Work

1.  Read `tasks/QUEUE.md`.
2.  If there are tasks in the "Ready" column, move the top one to "In Progress".
3.  Execute the task.
4.  If the task is completed, move it to "Done".
5.  If you get stuck, move it to "Blocked" and leave a comment.
6.  If the queue is empty, reply HEARTBEAT_OK.
