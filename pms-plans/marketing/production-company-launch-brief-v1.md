# Production Company Launch Brief (v1)

Date: 2026-02-28
Owner: Jordan + Aden
Status: Draft v1

## 1) Purpose
Launch a nimble production company focused on high-quality live events and media projects with disciplined operations from day one.

## 2) Positioning
A creative-first, execution-strong studio for:
- Live theater productions
- Concert/event programming
- Small-to-mid budget film/media projects

### Brand promise
"Beautifully executed productions with clear communication, professional process, and audience impact."

## 3) Initial Service Lines (90-day scope)
1. **Live Production Services**
   - concept development, staging support, run-of-show design
2. **Creative Direction + Performance Development**
   - performer coaching, repertoire/performance shaping
3. **Production Management Retainer**
   - scheduling, vendor coordination, show logistics

## 4) ICP (Ideal Client Profile)
- Independent artists and collectives
- Small venues/promoters
- Community arts organizations
- Emerging producers needing professional execution support

## 5) Offer Ladder (simple)
- **Starter:** one-off production consult / show blueprint
- **Core:** end-to-end production management package
- **Premium:** creative direction + production management + postmortem optimization

## 6) 90-Day Execution Plan

### Phase A (Weeks 1–2): Foundation
- Finalize name + identity
- Define service menu + pricing guardrails
- Build one-page capability deck
- Set up lead tracker + pipeline board

### Phase B (Weeks 3–6): Market Entry
- Outreach to first 30 warm/cold prospects
- Publish 3 credibility assets (case-style posts, process posts)
- Run 5 discovery calls
- Close first 1–2 paid engagements

### Phase C (Weeks 7–12): Delivery + Repeatability
- Deliver first projects with tight postmortems
- Collect testimonials + before/after outcomes
- Refine SOPs for pre-pro, production week, and strike

## 7) Core KPIs (first 90 days)
- Discovery calls booked: 10+
- Paid projects closed: 2–4
- Revenue target: defined baseline + stretch target
- On-time milestone delivery rate: 95%+
- Client satisfaction (post-project): 8/10+

## 8) Operational Backbone
- **Projects/tasks:** Notion board (pipeline + delivery)
- **Calendar:** Google Calendar (all commitments)
- **Knowledge/documents:** OpenLoops + Obsidian
- **Weekly rhythm:** Monday planning, Friday review/postmortem
