# Brand Quick Reference (PMS)

## Name
- Use: **Property Management Suite (PMS)**

## Voice
- Clear, direct, professional
- Outcome-first copy

## Logo
- Source: `Branding/Modern_Logo_1.png`
- Keep aspect ratio
- Min width: 120px digital / 25mm print
- Maintain clear space = icon height

## Core colors
- Deep Navy `#0B1220`
- Electric Blue `#2F80FF`
- Teal `#13C2C2`
- Success `#10B981`
- Warning `#F59E0B`
- Danger `#EF4444`

## Type
- Inter (headings + body)
- JetBrains Mono (IDs/code)

## UI rules
- Semantic status colors only
- High contrast (WCAG AA)
- Prefer explicit action labels

## CTA examples
- Start Application
- Save Payment Method
- Enable Autopay
- Request More Information

## Don’ts
- Don’t stretch/recolor logo
- Don’t use low-contrast text
- Don’t use inconsistent product naming
