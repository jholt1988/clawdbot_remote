# Alternate Brand Quick Reference (PMS)

## Direction name
**Modern Ledger** — premium, calm, operations-first.

## When to use
- Executive-facing decks
- Enterprise landing pages
- Product surfaces where trust and clarity matter more than visual energy

## Core tokens
- Ink 900: `#101828`
- Ink 700: `#344054`
- Canvas: `#F8F9FB`
- Paper: `#FFFFFF`
- Line: `#D0D5DD`
- Trust Navy: `#0A2540`
- Action Blue: `#1D4ED8`
- Signal Amber: `#FFC107` (sparingly)

## Typography
- Headlines: **Manrope 600/700**
- Body/UI: **Inter 400/500**
- Mono/data: **JetBrains Mono 400/500**

## Logo rules
- Source: `Branding/Modern_Logo_1.png`
- Keep aspect ratio, no redraws/effects
- Clear space: `1×` logomark height
- Min size: `120px` digital / `25mm` print
- On dark surfaces, use a light container if reverse logo variant is unavailable

## Component language
- 8px card radius, subtle borders, minimal shadow
- Primary button: Trust Navy bg / white text
- Secondary button: white bg / line border / Ink text
- Keep labels explicit; avoid icon-only critical actions

## Imagery and icons
- Architecture + operations scenes, lightly desaturated
- Outline icon set, 1.75–2px stroke
- Status colors only for real state semantics

## Motion tone
- Duration: `120–220ms`
- Easing: `cubic-bezier(0.2, 0.8, 0.2, 1)`
- Subtle fades/translates; no playful bounce in core product flows

## Messaging tone
- Outcome-first, evidence-led, low hype
- Example: “Automate routine approvals with full audit visibility.”

## Assumptions
1. Existing logo binary remains canonical.
2. No dedicated reverse/dark logo file currently in `Branding/`.
3. Alternate direction is additive (theme-based), not a rebrand.
