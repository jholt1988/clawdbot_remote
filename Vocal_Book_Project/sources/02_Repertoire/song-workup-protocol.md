# Song Workup Protocol

Purpose: Standard process for turning a new song choice into a performance-ready repertoire item.

## Trigger phrase
`new song: <title> by <artist>`

## Workflow
1. Create a new folder (or note set) for the song.
2. Duplicate `song-breakdown-template.md` and fill it fully.
3. Produce three outputs:
   - Song brief
   - Drill sheet
   - 7-day micro plan
4. Add task entries to Notion:
   - Analyze song
   - Warm-up/drill sessions
   - Performance simulation
   - Review and lock version
5. Schedule practice blocks on calendar (if requested).

## Required sections to complete every time
- Musical breakdown
- Lyric/interpretation map
- Vowel strategy
- Resonance analysis (required)
- Technical risk map
- Warm-up sequence
- Tough-area exercises
- Rehearsal ladder

## Exercise design rules
- Tie each exercise to a specific line/moment.
- Start at easier coordination (SOVT, hum, lip trill, NG) before full lyric.
- Use tempo progression: 60% → 75% → 90% → 100%.
- Keep drills short, specific, repeatable.
- Include at least one resonance-placement drill for each major hard section.

## Quality gate (definition of done)
A song is “rep-ready” when:
1. Hard moments have at least 2 successful clean takes.
2. Breath plan is stable across full run.
3. Vowel strategy is consistent on sustained/high notes.
4. Interpretation choices are intentional and repeatable.
5. One full performance take is recorded and reviewed.

## Output naming convention
- `song-brief-<slug>.md`
- `drill-sheet-<slug>.md`
- `7-day-plan-<slug>.md`

Example slug: `the-show-must-go-on`

## Optional scoring rubric (1–5)
- Pitch stability
- Breath management
- Diction clarity
- Emotional communication
- Stamina/endurance

Log scores each week to track growth.
